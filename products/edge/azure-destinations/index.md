# Azure


You can send data from Cribl Edge to the following Azure services:

| Destination | Description |
| --- | --- |
| [Azure Blob Storage](destinations-azure-blob) | Deliver data to Azure Blog Storage or Azure Data Lake Storage Gen2 |
| [Azure Data Explorer](destinations-azure-data-explorer) | Send batched or streaming data to the Azure Data Explorer managed data analytics service |
| [Azure Event Hubs](destinations-azure-event-hubs) | Send data to Azure Event Hubs |
| [Azure Monitor Logs](destinations-azure-monitor-logs) | Send data to Azure Monitor Logs |
| [Microsoft Sentinel](destinations-sentinel) | Send Log and Metric Events to Microsoft Sentinel SIEM |
