# Cribl Edge on Linux


Learn about deploying and configuring Cribl Edge on Linux.

---

Cribl Edge on Linux offers easy-to-use tools for exploring and collecting events.
You can run Cribl Edge on a supported Linux and collect events, logs, and metrics
via the full offering of Cribl Edge Sources.
