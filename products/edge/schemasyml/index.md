# schemas.yml


`schemas.yml` stores configuration data for the [Knowledge > Schema Library](schema-library).

```yaml {title="schemas.yml"}
# Schema ID - Unique identifier for the schema
# [string; required]
id:
# Description - Description of the schema
# [string]
description:
# Schema - JSON schema matching standards of draft version 2019-09
# [string; required]
schema:
# Tags - Tags associated with the schema
# [string]
tags:
```
