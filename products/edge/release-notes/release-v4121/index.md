# Cribl Edge 4.12.1


This release addresses a critical issue found in Cribl Stream, where Collection jobs accumulate obsolete internal files on the machines hosting Stream Workers. This issue does not affect Cribl Edge Nodes. See the [Cribl Edge 4.12.0](https://docs.cribl.io/edge/release-notes/release-v4120/) release notes for the latest feature and correction updates.
