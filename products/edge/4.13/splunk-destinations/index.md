# Splunk


You can send data from Cribl Edge to the following Splunk services:

| Destination | Description |
| --- | --- |
| [Splunk HEC](destinations-splunk-hec) | Stream data to a Splunk HEC receiver |
| [Splunk Single Instance](destinations-splunk) | Stream data to a Splunk instance |
| [Splunk Load Balanced](destinations-splunk-lb) | Load-balance data streaming to multiple Splunk receivers |
