# Cribl Edge in Containers


You can deploy Cribl Edge in containerized environments using Docker or Kubernetes.

Running Cribl Edge in a container ensures consistent deployment, simplifying installation and upgrades.
Containerization enables easy scaling and management via orchestration tools like Kubernetes.
