# Prometheus


You can send data from Cribl Edge to the following Prometheus services:

| Destination | Description |
| --- | --- |
| [Prometheus](destinations-prometheus) | Send metric events to Prometheus remote write targets |
| [Grafana Cloud](destinations-grafana_cloud) | Send data to Loki for logs and Prometheus for metrics |
| [Loki](destinations-loki) | Send log events to Loki |
