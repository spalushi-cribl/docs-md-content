# Prometheus


You can use the following [Sources](sources) to receive data from Prometheus services.

| Source | Description |
| --- | --- |
| [Prometheus Scraper](sources-prometheus) | Receive batched data from Prometheus targets |
| [Prometheus Remote Write](sources-prometheus-remote-write) | Receive metric data from Prometheus via the remote write protocol |
| [Grafana](sources-grafana) | Receive metric and log data from Grafana Agent via Prometheus remote write |
| [Loki](sources-loki) | Receive log data from Grafana Loki |
