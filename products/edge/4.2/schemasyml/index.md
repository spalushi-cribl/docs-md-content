# schemas.yml


`schemas.yml` stores configuration data for the [Knowledge > Schema Library](schema-library).

```yaml {title="$CRIBL_HOME/default/cribl/schemas.yml"}
schema_id: # [object] 
  description: # [string] Description - Brief description of this schema. Optional.
  schema: # [string] Schema - JSON schema matching standards of draft version 2019-09.
```
