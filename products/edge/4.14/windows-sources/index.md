# Windows


You can use the following [Sources](sources) to receive data from Windows services.

| Source | Description |
| --- | --- |
| [Windows Event Forwarder](sources-wef) | Receive events from Windows platforms |
| [Windows Event Logs](sources-windows-event-logs) | Read events from the Windows Events API |
| [Windows Metrics](sources-windows-metrics) | Collect metrics from a Windows host, and populate dashboards |
