# start


Starts Cribl Edge. Executes immediately. Upon first run, echoes the Cribl Edge default login credentials.

**Usage:**

`./cribl start`

**Sample response:**

```text
Starting...
...
Cribl started
```

**Arguments:**

| Option | Definition |
| ------ | ---------- |
| `-d <dir>` | Configuration directory. |
| `-r <role>` | Process role. |
