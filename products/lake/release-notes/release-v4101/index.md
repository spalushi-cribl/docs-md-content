# Cribl Lake 4.10.1


This release contains a number of enhancements for [Cribl Stream](/stream/release-notes/release-v4101),
[Cribl Edge](/edge/release-notes/release-v4101), and [Cribl Search](/search/release-notes/release-v4101).

Additionally, it provides the following correction for Cribl Lake:

## Corrections

When configuring a Cribl Lake Dataset, you will now be correctly notified
that accelerating fields whose names start with an underscore (for example, `_raw`), is not permitted.
