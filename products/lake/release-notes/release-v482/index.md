# Cribl Lake 4.8.2


## New Dataset Management Workflow

The workflow for creating and managing Datasets is improved, with a new UI,
including an extended set of filters to narrow down Dataset selection.

![Dataset list in Cribl Lake with filters at the top](new-lake-datasets-4.8.2.png)
{caption="New filters allow searching Datasets by connected Collectors or Destinations, and by retention period." border="true"}

Whenever you edit a Dataset, you can see a more detailed history of all modifications done to the Dataset.
