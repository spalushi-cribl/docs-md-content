# Grok Patterns


Use the built-in library of Grok patterns, or define your own.

---

A Grok pattern is a string of special characters
and regular expressions that match data. Cribl Search uses Grok patterns in definitions for [Parsers](parsers).

Cribl Search ships with many commonly used Grok patterns. The **Grok Patterns** page provides an interface for creating
and editing them.

To open the **Grok Patterns** page, select **Knowledge** > **Grok Patterns**.

## Add a Grok Pattern

1. On the top right, select **Add Grok Pattern File**.
1. Give your custom Grok pattern a unique **File name**.
1. Enter your Grok pattern in the **Content** pane.
1. Select **Save**.
