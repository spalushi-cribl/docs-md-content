# Connect Cribl Search to Google


Link Cribl Search to your Google services.

---

## Google Cloud Storage {#gcs}

Configure Cribl Search to query your Google Cloud Storage data. See:
[Google Cloud Storage](set-up-google-cloud-storage).

## Google Cloud Platform API {#gcp-api}

Configure Cribl Search to query a Google Cloud Platform API endpoint. See:
[Google Cloud Platform API](set-up-google-cloud-platform).

## Google Workspace API {#workspace-api}

Configure Cribl Search to query a Google Workspace API endpoint. See:
[Google Workspace API](set-up-google-workspace-api).
