# exp2


The `exp2` function calculates the base-2 exponential function of x, which is 2 raised to the power x: `2^x`.

## Syntax

    `exp2( X )`

### Arguments

* **X**: A real number, value of the exponent.

## Results

Exponential value of **X**.

## Examples {#examples}

This example returns `16`:

```kusto {runSearch=true}
print exp2( 4 )
```
