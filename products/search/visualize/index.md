# Visualize Search Results


Visualize your Cribl Search results with Charts, Tables, Dashboards, and other visualization tools.

---

## Charts

When you run an aggregate search, your results display on a Chart. Learn how to use Charts in the following sections:

- [Charts](charting): Learn the basics, export a Chart as an image, save and reset Chart settings.
- [Chart Types](chart-types): See all available types of Charts, and learn how to customize each of them.
- [Tables](results-table-settings): Learn how to format the results table that displays below every Chart.

## Dashboards

Cribl Search offers customizable Dashboards with a variety of visualization panels that you can tailor to your needs.
Learn how to use Dashboards in the following sections:

- [Dashboards](dashboards): Get an overview and learn about built-in sample Dashboards.
- [Create a Dashboard](creating-adding-to-dashboards): Learn different ways of creating a Dashboard, and how to fill it
  with visualizations.
- [Edit a Dashboard](editing-dashboards): Add, edit, hide, and arrange the visualization panels of your Dashboard.
- [Apply a Parent Search](editing-dashboards#parent): Aggregate results from multiple searches into a single
  visualization.
- [Add Inputs](dashboard-inputs): Link visualizations to specific data sources, using inputs.
- [Add Interactions](dashboard-interactions): Enable Dashboard viewers to drill down into specific values in a
  visualization panel.
- [Add Markdown](editing-dashboards#add-markdown): Enrich your Dashboards with custom Markdown panels.
- [Refresh Your Dashboards Automatically](editing-dashboards#autorefresh): Run a Dashboard on schedule, or configure its
  caching.
- [Add Visualizations Using Cribl Copilot](copilot-dashboards): Use the Cribl Copilot AI assistant to generate Dashboard
  visualizations.
- [Manage and Share Dashboards](managing-sharing-dashboards): Collect, export, and share your Dashboards.

## Third-Party Visualization Tools

You can easily visualize your Cribl Search results in Grafana, using the official Cribl Search plugin. See:
[Use Cribl Search with Grafana](grafana).
