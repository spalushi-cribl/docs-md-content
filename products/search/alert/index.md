# Send Alerts With Cribl Search


Receive conditional Notifications about your results, using scheduled searches and Notification targets.

---

## Scheduled Searches

[Schedule a search](scheduled-searches) to run at a specific frequency.

## Notifications

Set up a [Notification target](notifications) to send alerts about your scheduled searches to:




- System Messages: Banner messages in the Cribl Search UI (the default target).
- [Email](email-notifications) messages.
- [Amazon SNS](aws-sns-notification-targets) topics.
- [PagerDuty](pager-duty-notification-targets) accounts.
- [Slack](slack-notification-targets) channels.
- [Webhook](webhook-notification-targets) connections.
