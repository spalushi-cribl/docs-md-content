# Statements


A list of special statements supported by Cribl Search.

---

A statement in Cribl Search is a special keyword that sets options ([`set`](set)) or assigns names to expressions
([`let`](let)).

| Name         | Description                                                                                 |
| ------------ | ------------------------------------------------------------------------------------------- |
| [`let`](let) | Assigns a referenceable name to a value, expression, or an entire subquery.                 |
| [`set`](set) | Configures [options](set#available-set-statement-options) that affect one or more searches. |
