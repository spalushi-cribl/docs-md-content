# Cribl Search 4.4.2


## Dataset Providers

In [Dataset Providers](basic-concepts#dataset-providers) that support Splunk DDSS, you can now quickly configure DDSS by providing the root path for Splunk buckets. This applies to the following Data Provider types:
* [Amazon S3](set-up-s3)
* [Azure Blob Storage](set-up-azure-blob)
* [Cribl Edge](set-up-edge)
* [Google Cloud Storage](set-up-google-cloud-storage)
