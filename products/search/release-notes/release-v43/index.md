# Cribl Search 4.3


## New Features

### Query Language

The new [print operator](print) lets you quickly verify an expression before including it in a larger query.

The [lookup operator](lookup) now supports CIDR (Classless Inter-Domain Routing) for IP addresses and regular expressions for pattern matching.

### Enhanced User Experience

The [History table](search-overview#history) now offers customizable time selections, allowing you to control the displayed time frame.

You can now open any [Dashboard](dashboards) panel in a full-screen view.
