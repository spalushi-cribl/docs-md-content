# gamma


The `gamma` function computes [gamma function](https://en.wikipedia.org/wiki/Gamma_function).

## Syntax

    `gamma( X )`

### Arguments

* **X**: Parameter for the gamma function.

## Results

Gamma function of **X**.

## Examples {#examples}

This example returns `6`:

```kusto {runSearch=true}
print gamma( 4 )
```
