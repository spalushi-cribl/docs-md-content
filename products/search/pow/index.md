# pow


The `pow` function returns a result of raising to power.

## Syntax

    `pow( Base, Exponent )`

## Arguments

* **Base**: Base value.
* **Exponent**: Exponent value.

## Results

Returns base raised to the power exponent: **Base** ^ **Exponent**.

## Examples {#examples}

This returns `387,420,489`:

```kusto {runSearch=true}
print pow( 9,9 )
```
