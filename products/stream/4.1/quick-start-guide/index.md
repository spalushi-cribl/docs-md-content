# Quick Start Guide



## Quick Start Guide

Download latest version at https://cribl.io/download/. Install by uncompressing the file in the desired directory. Default location: /opt/cribl/.

> ##### Installation Requirements:
>
> * **OS**: 
>    * Linux: RedHat, CentOS, Ubuntu, AWS Linux, Suse (64bit)
>    * macOS 10.13 and 10.14+
> * **System**: 
>    * +4CPUs, +4GB RAM 
>    * 5GB free disk space
> * **CriblÂ Stream needs these ports to be available by default**:  
>   * UI Default: `9000`
>   * HTTP In Default: `10080`
>   * Worker communications: `4000`
>   * Phone home telemetry if required by license: `8000` to phonehome.cribl.io (34.220.85.61)
> 
> Node <version?>
> Git  1.8.3.1 (minimum)
>
