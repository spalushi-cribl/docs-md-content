# Data Lakes


You can send data from Cribl Stream to the following data lakes:

| Destination | Description |
| --- | --- |
| [Cribl Lake](destinations-cribl-lake) | Send data to Cribl Lake |
| [Amazon S3](destinations-data-lake-s3) | Send Cribl Search-formatted data to Amazon S3 |
| [Amazon Security Lake](destinations-security-lake) | Send OCSF data to Amazon Security Lake in Parquet format |
