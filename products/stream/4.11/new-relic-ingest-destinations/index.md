# New Relic Ingest


You can send data from Cribl Stream to the following New Relic Ingest services:

| Destination | Description |
| --- | --- |
| [New Relic Events](destinations-newrelic-events) | Send events to New Relic via the New Relic Event API |
| [New Relic Logs & Metrics](destinations-newrelic) | Send events to New Relic Log API and Metric API |
