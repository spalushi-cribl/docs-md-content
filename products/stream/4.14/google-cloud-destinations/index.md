# Google Cloud


You can send data from Cribl Stream to the following Google Cloud services:

| Destination | Description |
| --- | --- |
| [Google SecOps](destinations-google_chronicle) | Send data to Google Security Operations (SecOps) |
| [Google Cloud Chronicle API](destinations-google-chronicle-api) | Send to Google Cloud Chronicle API |
| [Google Cloud Logging](destinations-google-logging) | Send data to Google Cloud Logging |
| [Google Cloud Pub/Sub](destinations-google_pubsub) | Send data to Google Cloud Pub/Sub |
| [Google Cloud Storage](destinations-google-cloud-storage) | Send data to Google Cloud Storage |
