# Internal


You can send data from Cribl Stream to the following internal receivers:

| Destination | Description |
| --- | --- |
| [Cribl HTTP](destinations-cribl-http) | Send data between peer Nodes via HTTP |
| [Cribl TCP](destinations-cribl-tcp) | Send data between peer Nodes via raw TCP |
| [Default](destinations-default) | Specify a default output from among your Destinations |
| [DevNull](destinations-devnull) | Drop events when testing Pipelines and Routes |
| [Output Router](destinations-output-router) | Routes data to Destinations based on defined rules |
