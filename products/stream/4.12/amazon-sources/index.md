# Amazon


You can use the following [Sources](sources) to receive data from Amazon services.

| Source | Description |
| --- | --- |
| [Amazon Data Firehose](sources-kinesis-firehose) | Receive Amazon Data Firehose streams via HTTP endpoint |
| [Amazon Kinesis Data Streams](sources-kinesis-streams) | Receive data records from Amazon Kinesis Data Streams |
| [Amazon S3](sources-s3) | Receive data from Amazon S3 buckets |
| [Amazon Security Lake](sources-security-lake) | Configuring Amazon Security Lake Source |
| [Amazon SQS](sources-sqs) | Receive events from Amazon Simple Queuing Service |
