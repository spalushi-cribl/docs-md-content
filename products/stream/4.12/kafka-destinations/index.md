# Kafka


You can send data from Cribl Stream to the following Kafka services:

| Destination | Description |
| --- | --- |
| [Kafka](destinations-kafka) | Send data to a Kafka topic |
| [Confluent Cloud](destinations-confluent) | Send data to Kafka topics on Confluent Cloud |
| [Amazon MSK](destinations-msk) | Send data to a topic in Amazon MSK |
