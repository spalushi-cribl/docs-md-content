# Custom Login Page


You can customize the look of the login page for all Cribl apps.

> Custom login pages are available only in on-prem installations.
{.box .info}

## Configure Custom Login Page

To configure a custom login page:

1. In the CriblÂ Stream sidebar, select **Settings**, then **Global**.
1. Under **System**, select **General settings**.
1. Go to **Custom login page** and toggle **Enabled** to `Yes`.
1. In **Logo file**, select an image file that wll serve as a logo.
   The image must be no larger than 350x350px and under 2MB.
1. In **Content**, edit the provided HTML to modify the text block displayed under the login box.
