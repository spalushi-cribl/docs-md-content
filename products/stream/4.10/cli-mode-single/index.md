# mode-single


Configures instance as a Single-instance deployment.

**Usage:**

```text
./cribl mode-single -d d0e301e4-f961-4f50-956a-5153224b34e3
```

**Arguments:**

| Option | Definition |
| ------ | ---------- |
| `-d <deploymentId>` | Deployment ID for reporting telemetry on multiple deployments. |
