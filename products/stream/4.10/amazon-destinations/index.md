# Amazon


You can send data from Cribl Stream to the following Amazon services:

| Destination | Description |
| --- | --- |
| [Amazon CloudWatch Logs](destinations-cloudwatch-logs) | Send data to Amazon CloudWatch Logs |
| [Amazon Kinesis Data Streams](destinations-kinesis-streams) | Deliver data to an Amazon Kinesis Data Stream |
| [Amazon S3 Compatible Stores](destinations-s3) | Send data to Amazon S3 or to an S3-compatible store |
| [Amazon SQS](destinations-sqs) | Send events to Amazon Simple Queuing Service |
