# Office 365


You can use the following [Sources](sources) to receive data from Office 365 services.

| Source | Description |
| --- | --- |
| [Office 365 Activity](sources-office-365-activity) | Receive data from the Office 365 Management Activity API |
| [Office 365 Message Trace](sources-office365-msg-trace) | Receive Office 365 Message Trace data |
| [Office 365 Services](sources-office-365-services) | Receive data from the Microsoft Graph service communications API |
