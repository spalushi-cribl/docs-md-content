# Troubleshooting


Use the resources in this section to diagnose and resolve issues in Cribl Stream:

* [Known Issues](known-issues): Get information about issues that affect Cribl Stream, along with temporary workarounds and fix versions when applicable.
* [Diagnosing Issues](diagnosing): Create and export diagnostic bundles to share with Cribl Support.
* [Common Errors and Warnings](common-errors): Learn how to resolve common errors and warning messages in Cribl Stream.
* [Git Push Errors](git-push-errors): Learn how to resolve common errors when pushing a `git` commit.
* [Git Remote Repos & Trusted CAs](git-certificate-errors): Learn how to resolve errors related to self-signed certificates when pushing or pulling a `git` commit.
* [Sample Logs for Login Scenarios](sample-login-logs): Get examples of the event series that Cribl Stream records in common login scenarios.
